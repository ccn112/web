# THEME & LAYOUT GUIDE

## Theme

- Kế thừa token hiện tại.
- Nền sáng xanh nhạt.
- Grid mảnh.
- Navy heading.
- Blue/cyan accent.
- Gold chỉ cho CTA/active.
- Card trắng, border nhạt, shadow mềm.

## Title

- Tối đa 2 dòng ở desktop.
- `text-wrap: balance`.
- Không dùng heading quá dài.
- Mobile có thể 3 dòng nếu cần.

## Visual fallback

Nếu thiếu illustration:
- dùng SVG architecture;
- dùng card/layer/timeline;
- dùng icon existing;
- dùng radial glow;
- không chèn placeholder ảnh thô.
