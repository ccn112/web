## Mục tiêu checkpoint
Khóa baseline sau 5 set đầu tiên để các set tiếp theo (FinERP Enterprise Operations, XBuilding, X.AI, X.Space...) tích hợp trên cùng chuẩn mà không làm lệch cấu trúc.

## Lưu ý
- SET 06 là gói 6 ảnh, không phải 8 ảnh.
- Một số package tham chiếu (UI Style, Workflow Website) được để ở `packages/reference_optional/` để Claude tham khảo thêm, không phải thành phần bắt buộc của checkpoint.
- Chuẩn tỷ lệ mặc định: 4:3.
- Không dùng nhãn nội bộ kiểu SET XX / CODE trong ảnh.
