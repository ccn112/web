# C05 SECTION SPEC

## C05-01 — Solution Suite Selector

**Title:** Một hệ sinh thái, cấu hình theo doanh nghiệp

**Purpose:** Điều hướng người dùng chọn đúng mô hình doanh nghiệp.

**Desktop layout:** copy trái, visual selector phải.

**Mobile:** visual trên, 5 lựa chọn dạng accordion/cards.

**Data:** `solutionSuites`.

---

## C05-02 — Solution Blueprint

**Title:** Một blueprint cho toàn bộ hành trình

4 lớp:

1. Hành trình nghiệp vụ
2. Sản phẩm tham gia
3. Nền tảng dùng chung
4. Dịch vụ triển khai
