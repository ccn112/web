# QA ACCEPTANCE CHECKLIST

## Nội dung
- [ ] Không lặp lại nội dung sản phẩm/giải pháp một cách máy móc.
- [ ] Trang trả lời được: chọn gì, cấu hình gì, bắt đầu từ đâu.
- [ ] Title section tối đa 2 dòng.
- [ ] Toàn bộ text tiếng Việt.
- [ ] Không có số liệu giả.

## Theme
- [ ] Bám token hiện tại.
- [ ] Nền grid sáng đồng bộ.
- [ ] Gold accent tiết chế.
- [ ] Section ảnh trái/phải luân phiên.
- [ ] Không tạo theme phụ xa lạ.

## CMS
- [ ] Suite và nội dung quản trị được từ CMS.
- [ ] Preview và revalidate hoạt động.
- [ ] Route slug được khóa.
- [ ] Media có alt/focal point.
- [ ] Form submission được lưu.

## Responsive
- [ ] 1440.
- [ ] 1024.
- [ ] 768.
- [ ] 390.
- [ ] Heading không vượt 2 dòng ở desktop.
- [ ] Không overflow ngang ngoài carousel/matrix có chủ đích.

## Accessibility
- [ ] Tab/selector ARIA.
- [ ] Keyboard focus.
- [ ] Form labels rõ ràng.
- [ ] Error messages đọc được.
- [ ] Reduced motion.

## Performance
- [ ] Visual raster dùng WebP/AVIF.
- [ ] Below-fold lazy load.
- [ ] Không render reference PNG.
- [ ] Không client hóa toàn trang.
