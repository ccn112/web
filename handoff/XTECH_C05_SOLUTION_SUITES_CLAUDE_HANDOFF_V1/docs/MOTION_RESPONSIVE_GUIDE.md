# MOTION & RESPONSIVE GUIDE

## Motion

- reveal 0.5–0.7s;
- stagger 60–90ms;
- connector draw 0.8–1.2s;
- hover lift tối đa 4px;
- active glow tăng nhẹ;
- roadmap progress có thể theo scroll;
- reduced motion bắt buộc.

## Desktop

- max content 1280–1320px;
- split layout 44/56 hoặc 50/50;
- heading tối đa 2 dòng;
- matrix full-width.

## Tablet

- split thành 1 cột;
- visual xuống dưới;
- configurator dùng horizontal scroll có kiểm soát.

## Mobile

- selector thành accordion;
- blueprint thành 4 block dọc;
- scope cards thành swipe/stack;
- matrix chuyển sang từng suite, không giữ bảng rộng;
- roadmap stepper dọc;
- builder form 1 cột;
- recommendation xuất hiện sau submit.
