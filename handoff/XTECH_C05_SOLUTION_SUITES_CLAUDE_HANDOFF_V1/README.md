## Theme

Bám tương đối theme hiện tại của website XTECH:

- nền sáng xanh rất nhạt;
- grid mảnh;
- title navy/blue gradient;
- gold accent dùng tiết chế;
- card trắng bo góc lớn;
- visual 3D/hub dùng như asset chính;
- section luân phiên ảnh trái / ảnh phải;
- tiêu đề section tối đa 2 dòng.
