# 06 — Style Rule for Future Sets

## Quy tắc đã chốt cho các bộ tiếp theo

1. Không chèn nhãn kiểu:
   - "SET 04 / SUITE-08"
   - "SET 05 / XBK-03"
   - hoặc các mã batch tương tự vào trong ảnh.

2. Ảnh cần tự nhiên hơn:
   - giữ các label nội dung ngắn nếu cần để dễ hiểu;
   - bỏ các nhãn quản trị nội bộ của quá trình sản xuất.

3. Tiêu đề chính trên website phải render bằng HTML.
4. Trong ảnh chỉ nên có:
   - nhãn ngắn mô tả thành phần;
   - tên lớp / tên module khi thật sự cần.
5. Font tiêu đề định hướng: Plus Jakarta Sans.
6. Tông màu giữ theo nhận diện XTECH.
