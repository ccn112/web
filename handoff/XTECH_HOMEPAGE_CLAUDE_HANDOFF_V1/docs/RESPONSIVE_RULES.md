# RESPONSIVE RULES

## >= 1440
- Max content 1320px.
- Hero 820–920px.
- Headline 72–88px.
- Hub/orbit đủ chiều sâu.

## 1024–1439
- Max content 1120px.
- Orbit giảm 15–20%.
- Tabs 3 cột.
- Value cards 3 cột.

## 768–1023
- Split layout chuyển một cột.
- Product orbit nằm dưới copy.
- Timeline có thể 3x2 hoặc scroll ngang.
- Semantic visual giảm node.

## < 768
- Hero 680–780px.
- CTA stack/flex-wrap.
- Hub/orbit không thu nhỏ nguyên desktop: đổi carousel/accordion.
- Connector SVG phức tạp tắt.
- Text body tối thiểu 16px.
- Card một cột.
