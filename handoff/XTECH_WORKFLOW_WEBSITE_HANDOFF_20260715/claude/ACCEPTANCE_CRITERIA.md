# Acceptance Criteria

## Assets
- [ ] Seven workflow assets imported.
- [ ] Original PNG retained.
- [ ] WebP/AVIF generated.
- [ ] Logo comes only from `brand/`.
- [ ] Every asset has Vietnamese alt text.

## CMS and seed
- [ ] Seed can run twice without duplication.
- [ ] Six workflow stages exist.
- [ ] Corporate page is published.
- [ ] Menu item is added once.
- [ ] Product page blocks are patched once.

## Frontend
- [ ] Route returns 200.
- [ ] Stage navigation works.
- [ ] Anchor deep links work.
- [ ] Desktop alternates image position.
- [ ] Mobile navigation is usable.
- [ ] Lightbox opens original image.
- [ ] Product links are correct.

## Chatbot
- [ ] Current stage is included in context.
- [ ] Prompt chips change by stage.
- [ ] Chatbot does not invent unsupported features.

## SEO
- [ ] One H1.
- [ ] Canonical is correct.
- [ ] Page is in sitemap.
- [ ] Images have alt text.
- [ ] Internal links connect corporate and product sites.

## Quality
- [ ] Typecheck passes.
- [ ] Lint passes.
- [ ] Tests pass.
- [ ] No duplicate seed records.
- [ ] Lighthouse Accessibility >= 90.
