# QA & ACCEPTANCE CHECKLIST

## Visual
- [ ] Nhìn vào section nhận ra rõ bố cục reference.
- [ ] Không bị flatten thành card grid.
- [ ] Có main visual/hub/layer đúng hierarchy.
- [ ] Connector line sắc nét.
- [ ] Background có chiều sâu nhưng không lấn chữ.

## Content
- [ ] Toàn bộ chữ tiếng Việt.
- [ ] Heading/paragraph/CTA là HTML.
- [ ] Không dùng chữ nằm trong ảnh.
- [ ] Không dùng số liệu chưa xác thực.

## Brand
- [ ] Logo XTECH là asset gốc.
- [ ] Màu blue/cyan đồng bộ.
- [ ] Font Manrope/Inter.

## Responsive
- [ ] 1440px.
- [ ] 1024px.
- [ ] 768px.
- [ ] 390px.
- [ ] Không có overflow ngang ngoài vùng carousel.
- [ ] Mobile có composition riêng.

## Accessibility
- [ ] `aria-labelledby`.
- [ ] decorative image alt rỗng.
- [ ] keyboard focus.
- [ ] tabs/accordion ARIA đúng.
- [ ] reduced motion.

## Performance
- [ ] AVIF/WebP.
- [ ] next/image.
- [ ] width/height hoặc fill đúng.
- [ ] lazy load dưới fold.
- [ ] không dùng video/particle nặng.
