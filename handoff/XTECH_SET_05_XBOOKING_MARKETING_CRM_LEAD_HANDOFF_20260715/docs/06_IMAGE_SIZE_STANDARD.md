# 06 — Image Size Standard

## Khuyến nghị chuẩn cho các set tiếp theo

### Chuẩn chính: 4:3
Dùng cho phần lớn illustration asset trên website vì:
- cân bằng tốt giữa desktop và tablet;
- dễ dùng cho section split layout (text bên trái, ảnh bên phải);
- dễ crop để tạo thumbnail và preview;
- phù hợp cho cả corporate page lẫn product page.

### Khi nào dùng 16:9
Chỉ dùng 16:9 cho:
- hero rất rộng;
- banner hoặc cover cho bài viết / Open Graph;
- section cần cảm giác panorama hoặc horizontal flow.

### Khi nào dùng 4:4 (1:1 gần vuông)
Chỉ dùng 4:4 cho:
- social / thumbnail / card nhỏ;
- icon scene hoặc mini visual;
- ảnh dùng trong grid vuông.

## Kết luận
- **Mặc định cho các set nội dung website: 4:3**
- **Hero cover / OG / banner: 16:9**
- **Card nhỏ / social crop: 4:4**
