# CMS Seed Hints

## Mục tiêu seed
Tạo đủ dữ liệu để giao diện admin sau khi đổi theme có nội dung demo rõ ràng.

## Collections gợi ý (nếu repo có)
- Users
- Posts / News
- Categories
- Tags
- Pages
- Media
- Authors / Team (nếu có)

## Seed data tối thiểu
### 1. Users
- 1 super admin
- 1 editor

### 2. Categories / Tags gợi ý
Các tag nên liên quan hệ sinh thái phần mềm XTECH:
- XBooking
- FinERP
- XBuilding
- X.AI
- X.Space
- Chuyển đổi số
- Dữ liệu & AI
- Tự động hóa
- Tích hợp hệ thống
- Bất động sản
- Vận hành doanh nghiệp
- Quản trị dữ liệu
- AI Agent
- Cloud
- Workflow

### 3. Posts mẫu
- XTECH ra mắt hệ sinh thái AI cho doanh nghiệp
- 5 bước xây dựng nền tảng dữ liệu doanh nghiệp
- Chuyển đổi số bất động sản cần bắt đầu từ đâu?
- XBuilding và bài toán vận hành cư dân hiện đại
- Vì sao doanh nghiệp cần workflow + AI thay vì phần mềm rời rạc?

### 4. Pages mẫu
- Trang chủ
- Về XTECH
- Giải pháp
- Sản phẩm
- Tin tức
- Liên hệ

## Lưu ý
- Seed phải idempotent nếu có thể.
- Dùng slug, status, excerpt, tags để test UI listing/filter.
